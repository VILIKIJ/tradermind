🚀 TraderMind — Запуск на сервере

Следуй этим шагам по порядку. Займёт около 15 минут.

Шаг 1 — GitHub: создай репозиторий

1Зайди на GitHub

Открой в браузере:

github.com/new

Войди в свой аккаунт если нужно.

2Создай новый репозиторий

Repository name: напиши tradermind

Visibility: выбери Public

Всё остальное не трогай.

Нажми зелёную кнопку "Create repository"

3Загрузи файлы

На странице репозитория нажми "uploading an existing file" (ссылка посередине страницы)

Перетащи в окно ВСЕ три файла из папки tradermind-server:

📄 server.js
📄 package.json
📄 .gitignore

Затем создай папку public и загрузи в неё index.html

⚠️ Важно: файл .gitignore может быть скрыт на Mac. Включи показ скрытых файлов: Cmd+Shift+точка

Нажми "Commit changes" (зелёная кнопка внизу)

Шаг 2 — Render: запусти сервер бесплатно

4Зайди на Render
render.com

Нажми "Get Started for Free" → Sign up with GitHub

Разреши Render доступ к GitHub.

5Создай Web Service

На главной странице Render нажми "New +" → "Web Service"

Выбери "Build and deploy from a Git repository"

Нажми "Next"

В списке репозиториев найди tradermind и нажми "Connect"

6Настрой сервис

Заполни поля:

Name: tradermind

Region: Frankfurt (EU Central) — для скорости

Branch: main

Runtime: Node

Build Command: npm install

Start Command: node server.js

Instance Type: Free

7Добавь API-ключ Anthropic

Прокрути вниз до раздела "Environment Variables"

Нажми "Add Environment Variable"

Key: ANTHROPIC_API_KEY

Value: вставь свой ключ который начинается с sk-ant-...

⚠️ Ключ виден только тебе на сервере — он не попадёт в код

Нажми "Create Web Service"

8Подожди деплой (3-5 минут)

Render начнёт собирать приложение. Ты увидишь логи. Жди пока появится:

✅ Your service is live

Твоя ссылка будет выглядеть так:

https://tradermind.onrender.com

Шаг 3 — iPhone: добавь на рабочий стол

9Открой на iPhone

Скопируй ссылку с Render и открой её в Safari на iPhone.

⚠️ Именно Safari — Chrome на iPhone не поддерживает добавление на рабочий стол
10Добавь на экран Домой

Нажми кнопку Поделиться (квадрат со стрелкой вверх) внизу Safari

Прокрути список вниз и нажми "На экран «Домой»"

Название: TraderMind

Нажми "Добавить"

✅ Иконка появится на рабочем столе iPhone!

Открывается как настоящее приложение — без адресной строки браузера. Гуру работает. Данные сохраняются.

Если что-то пошло не так

Ошибка при деплое на Render: проверь что Build Command = npm install и Start Command = node server.js

Гуру не отвечает: проверь что ANTHROPIC_API_KEY добавлен правильно в Environment Variables на Render

Страница не открывается: подожди ещё 2-3 минуты — первый старт медленный

Бесплатный Render засыпает: первый запрос после паузы идёт 30 секунд — это нормально. Потом быстро.
